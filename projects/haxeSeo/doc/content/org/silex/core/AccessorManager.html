static function revealAccessors(toBeRevealed : String, accessorsList : Dynamic) : Dynamic

Takes toBeRevealed string and resolves its accessors using accessorsList. "Resolve" means to replace each accessor by its equivalent string value.

example1: if accessor is '<>' and accessorsList:{DataSelector1=>selectedItem=>title:"layer1",silex.config=>version:"v1.6.1"}, => result will be 'layer1' example2: "my text is for <>!!! not for <> ((nor for <>!!!))" becomes => "my text is for valueName1!!! not for valueName2 nor for valueName3!!!" if all name variables exist => "my text is for valueName1!!! not for valueName2" if only dbdata.name1 & dbdata.name2 exists => "" if only dbdata.name1 or dbdata.name2 exists => "nor for valueName3!!!" if only dbdata.name3 exists

@param toBeRevealed the string to be revealed, containing accessors ("((...<>...))") @param accessorsList the accessors list used to reveal the accessor @return Dynamic the resolved accessor, which can be a string, an array or a Hash
